Project Overview

Currently developing an AI-powered web application integrated with BuyRentRwanda.com, which recommends properties to users based on their preferences, browsing behavior, and real property data. The system leverages machine learning algorithms to provide personalized property suggestions for a better user experience.

Current Progress

  • Developing backend algorithms for personalized property recommendations
  • Integrating real property data dynamically into the AI system
  • Strengthening skills in Python, machine learning, and full-stack development
  • Working on performance optimization and scalable architecture

Technologies Used

Python, Machine Learning (ML), Pandas, NumPy, Flask/Django, JavaScript, HTML/CSS, Real Estate Data Integration

Outcome & Impact

This project aims to provide users with a personalized property search experience, improve engagement on BuyRentRwanda.com, and demonstrate practical skills in AI, data handling, and full-stack development.
